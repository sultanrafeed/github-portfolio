# GitHub Portfolio

A personal portfolio website for Rafeed Mohammad Sultan featuring a creative biriyani-inspired theme and a polished showcase of research, software projects, experience, and contact information.

## About

This web portfolio blends visual storytelling with professional accomplishments. It presents:

- Research papers and AI-focused publications
- Personal software projects and GitHub repositories
- Work experience, internships, and education
- A contact form for inquiries and connection requests

The homepage uses a custom interactive layout, audio feedback, animated cursor effects, a mock GitHub contribution graph, and a responsive contact section.

## Key Features

- `index.html`: main portfolio page with a modern hero layout and creative design
- `style.css`: custom styling with a warm biriyani-themed palette and responsive section design
- `script.js`: interactive cursor, sound effects, loading animation, clickable project cards, and GitHub graph rendering
- Downloadable CV link and external links to LinkedIn, GitHub, and email
- Contact form using FormSubmit for direct messaging

## Project Sections

1. Hero introduction with branding and themed visual elements
2. Research showcase with recent publications and AI research highlights
3. Project gallery featuring software apps, ML projects, and full-stack work
4. Experience & education timeline with roles and academic achievements
5. Contact form and social links for outreach

## How to View Locally

1. Clone or download the repository.
2. Open `index.html` in your browser.

> Tip: For the contact form to work correctly, host the site on GitHub Pages or a local development server instead of opening the file directly.

## Recommended Deployment

- GitHub Pages
- Netlify
- Vercel
